Gestión de temáticas y palabras clave

Activa, desactiva o edita las temáticas y términos que vigila el monitor. Al terminar, descarga el archivo y súbelo a GitHub.

📌 Temáticas toggle=activa/desactiva · editar=keywords
➕ Extras — términos sueltos

Palabras que no encajan en ninguna temática. Aparecen agrupadas como "Extras".

¿Cómo aplicar los cambios?

  • Edita temáticas y keywords aquí
  • Haz clic en ⬇ Descargar user_config.json
  • Entra a GitHub → carpeta docs/
  • Sube el archivo (reemplaza el existente)
  • El monitor lo leerá en la próxima ejecución

Consejos para las keywords

  • Busca en título, organismo y sección del BOE
  • No distingue mayúsculas
  • Puedes desactivar una keyword sin borrarla
  • Evita siglas muy cortas o ambiguas (ej: "ITV" puede dar falsos positivos)

⚡ Tras subir el archivo

Para reprocesar el historial con las nuevas keywords, lanza el workflow BOE Backfill manualmente desde GitHub Actions.
